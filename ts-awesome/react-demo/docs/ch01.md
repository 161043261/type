# ch01

JSX 规则

1. 组件（函数）只能返回一个根元素

```tsx
function Component() {
  return (
    <div>
      <h1>待办事项</h1>
      <ul>
        <li>唱</li>
        <li>跳</li>
        <li>rap</li>
      </ul>
    </div>
  );
}
```

或使用空标签 Fragment

```js
function Component() {
  return (
    <>
      <h1>待办事项</h1>
      <ul>
        <li>唱</li>
        <li>跳</li>
        <li>rap</li>
      </ul>
    </>
  );
}
```

2. 标签必须闭合

```tsx
function Component() {
  // <img> 等自闭合标签必须写为 <img />
  return <img src="vite.svg" alt="vite logo" />;
}
```

3. 属性名转换为驼峰命名

例：stroke-width => strokeWidth, class => className

### 插值字符串

```tsx
export default function Avatar() {
  const avatar = 'https://example.com/.jpg';
  const description = 'Gregorio Y. Zara';
  return (
    <img
      className="avatar"
      src={avatar}
      alt={description}
    />
  );
}

```
