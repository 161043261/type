# HTML

| 原字符 | 等价字符引用 |
| ------ | ------------ |
| <      | &lt;         |
| >      | &gt;         |
| "      | &quot;       |
| '      | &apos;       |
| &      | &amp;        |

- png: 无损压缩
- bmp: 不压缩

- header 头部
- footer 底部
- nav 导航
- article 文档
- section 区域
- aside 侧边栏
- main 主要部分
